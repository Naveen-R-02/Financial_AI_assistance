# Financial_AI_assistance
An AI-powered financial assistant designed to empower rural communities by explaining banking services, financial terms, and basic money management in English, Hindi, and Kannada languages, improving financial literacy and access to essential financial knowledge using easy-to-use technology.

This project was developed as part of an academic requirement using Python and Flask.

---

## Project Aim

The aim of this project is to provide clear and simple explanations about banking services such as accounts, loans, deposits, and basic financial terms.  
It supports multiple languages so that users can interact in a language they are comfortable with.

---

## Features

- Text-based financial query support  
- Voice input and voice output support  
- Multilingual responses (English, Hindi, Kannada)  
- Simple rule-based logic for answering questions  
- Easy-to-use web interface  

---

## How It Works

The user enters a question related to banking or finance through text or voice input.  
The system checks the keywords in the question and matches them with predefined financial information stored in the project.  
If a suitable match is found, the answer is displayed and can also be spoken using text-to-speech.
